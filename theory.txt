5. Статические свойства не привязаны к конкретному объекту и могут существовать без создания экземпляра и слова new. Чаще всего их применяют к функциям и классам

6.Геттеры и сеттеры позволяют определять специальные методы для доступа к объекту. get используется для получения значения свойства. set используется для установления значения для свойства.
 var person = {
   language : "en",
   get lang() {
      return this.language;
   }
};
document.getElementById("demo").innerHTML = person.lang; 

var person = {
    language : "",
    set lang(lang) {
      this.language = lang;
   }
};
person.lang = "en";
document.getElementById("demo").innerHTML = person.language; 

7. В этом скрипте создается объект, поэтому и тип выведет object
